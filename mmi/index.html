Minimal mutation-infinite quivers

Website providing accompanying material for a paper by John Lawson on the classification of all minimal muatation-infinite quivers.

This site gives a description of all the software used in the computations and calculations required to achieve the result, as well as a lookup page which illustrates all the classes of minimal mutation-infinite quivers.
